API Reference

▓▓ Core Commands

▒▒ osvm balance

osvm balance [ADDRESS] [OPTIONS]

OPTIONS:
  -C, --config PATH    Config file path
  -u, --url URL       RPC endpoint
  -v, --verbose       Verbose output

▒▒ osvm svm

osvm svm SUBCOMMAND

SUBCOMMANDS:
  list        List all SVMs
  get         Get SVM details
  dashboard   Interactive dashboard
  install     Deploy SVM to host

▒▒ osvm nodes

osvm nodes SUBCOMMAND

SUBCOMMANDS:
  list        List nodes
  status      Node status
  logs        View logs
  restart     Restart node
  stop        Stop node
  deploy      Deploy new node

▓▓ Advanced Features

▒▒ osvm audit

osvm audit [REPOSITORY] [OPTIONS]

OPTIONS:
  --output-dir DIR    Output directory
  --format FORMAT     json|html|markdown
  --no-commit         Don't commit results
  --gh-repo REPO      GitHub repository

▒▒ osvm mcp

osvm mcp SUBCOMMAND

SUBCOMMANDS:
  add         Add server
  list        List servers
  enable      Enable server
  disable     Disable server
  test        Test connectivity
  tools       List tools
  call        Execute tool

▒▒ osvm chat

osvm chat [OPTIONS]

OPTIONS:
  --advanced   FAR-style UI
  --microvm    Run in microVM
  --test       Run UI tests
  --debug      Debug mode
