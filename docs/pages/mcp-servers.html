Model Context Protocol (MCP) Servers

▓▓ MCP with Ephemeral MicroVMs

NEW: All MCP tools now execute in ephemeral microVMs by default!

[INFO] Each tool call:
  • Launches fresh microVM
  • Executes in complete isolation
  • Destroys VM after completion
  • Zero state persistence

▓▓ Quick Setup

osvm mcp setup

Quick setup for Solana MCP

▓▓ Server Management

osvm mcp list

List configured servers

osvm mcp add my-server stdio "node server.js"

Add stdio-based server

osvm mcp add-github solana https://github.com/solana/mcp-server

Add server from GitHub

▓▓ Tool Execution

osvm mcp tools solana-mcp

List available tools

osvm mcp call solana-mcp get_balance

Execute tool (runs in ephemeral VM)

▓▓ MicroVM Isolation

osvm mcp microvm launch my-server

Launch server in dedicated microVM

osvm mcp microvm status

Check microVM status
