Command Reference

Complete API documentation for OSVM CLI

Core Commands

# Balance Operations
osvm balance [ADDRESS]                    # Check SOL balance
  --url URL                               # Custom RPC endpoint
  --commitment LEVEL                      # Commitment level

# SVM Management
osvm svm list                            # List all SVMs
osvm svm get <NAME>                      # Get SVM details
osvm svm dashboard                       # Interactive dashboard
osvm svm install <NAME> <HOST>          # Deploy SVM

# Node Operations
osvm nodes list                          # List all nodes
  --network NETWORK                      # Filter by network
  --type TYPE                            # Filter by type
  --status STATUS                        # Filter by status
osvm nodes status <NODE-ID>             # Check node status
osvm nodes logs <NODE-ID> [--follow]    # View node logs
osvm nodes restart <NODE-ID>            # Restart node

Global Options

# Configuration
--config PATH         # Path to config file
--keypair PATH        # Path to keypair file
--url URL            # RPC URL override
--commitment LEVEL   # Commitment level (finalized|confirmed|processed)

# Output Control
--verbose, -v        # Verbose output (repeat for more)
--debug             # Enable debug mode
--json              # JSON output format
--no-color          # Disable colored output

# Help & Version
--help, -h          # Show help information
--version, -V       # Display version

Environment Variables

# Solana Configuration
SOLANA_CONFIG        # Path to Solana config file
SOLANA_RPC_URL      # Default RPC endpoint
SOLANA_KEYPAIR      # Default keypair path

# AI Configuration
OPENAI_KEY          # OpenAI API key
OPENAI_URL          # AI endpoint URL

# Debug Settings
RUST_LOG            # Logging level (debug|info|warn|error)
RUST_BACKTRACE      # Show backtraces (1|full)
NO_COLOR            # Disable colors (1)

Configuration File

# ~/.config/osvm/config.yml
json_rpc_url: https://api.mainnet-beta.solana.com
keypair_path: ~/.config/solana/id.json
commitment: confirmed
verbose: false
ai:
  provider: openai
  model: gpt-4
  temperature: 0.7
mcp:
  servers:
    - name: solana-mcp
      url: http://localhost:8080
      enabled: true

Exit Codes

0   Success
1   General error or critical security finding (audit)
2   Command line parsing error
3   Configuration error
4   Network/RPC error
5   Authentication/keypair error
10  Deployment failure
11  SSH connection failure
20  Self-repair failure

Advanced Features

# Self-repair system
$ osvm doctor                    # Run diagnostics
$ osvm doctor --fix              # Auto-repair issues

# Circuit breaker (automatic)
• Failure threshold: 5
• Success threshold: 2
• Timeout: 30s
• Reset timeout: 60s

# Hot-swap deployment
$ osvm deploy --hot-swap         # Zero-downtime update
• Blue-green deployment
• State preservation
• Automatic rollback

Example Workflows

# Complete validator deployment
$ osvm user@validator.example.com \
  --svm sonic \
  --node-type validator \
  --network mainnet \
  --hot-swap

# AI-assisted development
$ osvm "create a token program with freeze authority"
$ osvm audit ./program --ai
$ osvm deploy ./target/deploy/program.so --network devnet

# MCP-powered automation
$ osvm mcp setup
$ osvm chat --advanced
> "Monitor my validator and alert on issues"
