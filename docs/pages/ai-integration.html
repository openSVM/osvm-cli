AI Integration

▓▓ Natural Language Commands

OSVM understands natural language queries. Simply type what you want to do:

osvm "deploy my program to mainnet"

AI interprets and executes your intent

osvm "show me all validators on sonic"

Natural language query processing

▓▓ Interactive Chat

osvm chat

Launch basic chat interface

osvm chat --advanced

FAR-style multi-session interface

osvm chat --microvm

Isolated chat with ephemeral tool VMs

▓▓ AI-Powered Auditing

osvm audit github.com/project/contract

Security audit with AI analysis

▓▓ Configuration

▒▒ OpenAI

export OPENAI_URL="https://api.openai.com/v1/chat/completions"
export OPENAI_KEY="sk-your-key-here"

▒▒ Local AI (Ollama)

export OPENAI_URL="http://localhost:11434/v1/chat/completions"
export OPENAI_KEY="ollama"

▓▓ Circuit Breaker

OSVM includes automatic failure protection:

  • 5 failure threshold
  • 30 second timeout
  • Automatic recovery
  • Fallback to local processing
