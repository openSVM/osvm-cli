MCP Server Management

Connect and manage Model Context Protocol servers

🔌 Server Configuration

# Add a new MCP server
$ osvm mcp add my-server http://localhost:3000

# Add from GitHub
$ osvm mcp add-github solana-mcp \
  https://github.com/opensvm/solana-mcp

# Quick Solana setup
$ osvm mcp setup

🛠️ Tool Management

# List available tools
$ osvm mcp tools solana-mcp

Available tools:
• get_balance
• get_recent_transactions
• deploy_program
• stake_sol

⚡ Tool Execution

# Execute MCP tool
$ osvm mcp call solana-mcp get_balance \
  --address "Your...Address"

Executing tool...
✓ Balance: 100.5 SOL

Server Management Commands

# List configured servers
$ osvm mcp list

# Enable/disable servers
$ osvm mcp enable solana-mcp
$ osvm mcp disable custom-mcp

# Test server connectivity
$ osvm mcp test solana-mcp
✓ Server responding
✓ Authentication successful
✓ 15 tools available

Authentication Methods

# Bearer token authentication
$ osvm mcp add api-server https://api.example.com \
  --auth bearer --token "YOUR_TOKEN"

# Basic authentication
$ osvm mcp add secure-server https://secure.example.com \
  --auth basic --username user --password pass

# API key authentication
$ osvm mcp add key-server https://key.example.com \
  --auth api_key --token "API_KEY_HERE"
