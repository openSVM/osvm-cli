  --node-type validator \

🔄 Hot-Swap Updates

# Zero-downtime updates
$ osvm deploy --hot-swap

Blue-green deployment:
1. Deploy new version (green)
2. Sync state from blue
3. Switch traffic to green
4. Terminate blue instance

✓ No downtime
