🧠 OVSM Language

Open Versatile Seeker Mind - AI Agent Research Planning Language

OVSM (pronounced "OH-vism") is a domain-specific language for expressing AI agent research plans with conditional branching, parallel execution, and adaptive learning.

📋 Overview

OVSM enables AI agents to articulate sophisticated investigation strategies for blockchain research, statistical analysis, and knowledge discovery.

🔍 Research-Optimized

Built for multi-step investigation workflows with conditional branching and decision points.

🤖 Multi-Agent

Native support for agent delegation, coordination, and result merging.

📊 Statistical

Built-in hypothesis testing, causal inference, and confidence scoring.

🧠 Adaptive

Meta-learning from past queries to optimize future research strategies.

💡 Quick Example

## Q: "What is the average transaction fee on Solana?"

**Expected Plan:**

[TIME: ~15s] [COST: free] [CONFIDENCE: 95%]

**Available Tools:**
From Standard Library: getSlot, getBlock, MAP, MEAN, MEDIAN

**Main Branch:**
$current_slot = getSlot()
$blocks = []

FOR $i IN 0..10:
  $block = getBlock(slot: $current_slot - $i)
  $blocks = APPEND(array: $blocks, item: $block)

$fees = MAP($blocks, b => b.transactions).flat().map(tx => tx.fee)
$mean_fee = MEAN(data: $fees)

DECISION: Check distribution
  BRANCH A (high variance):
    $result = MEDIAN(data: $fees)
  BRANCH B (normal):
    $result = $mean_fee

**Action:**
RETURN {average_fee: $result, confidence: 95}

💻 Example: Multi-Agent Research

PARALLEL_AGENTS {
  $price_analysis = SPAWN_AGENT(
    agent_type: "price_analyzer",
    task: "Track price movements",
    context: {token: $token_address}
  )

  $volume_analysis = SPAWN_AGENT(
    agent_type: "volume_analyzer",
    task: "Analyze trading volume",
    context: {token: $token_address}
  )

  $sentiment = SPAWN_AGENT(
    agent_type: "sentiment_analyzer",
    task: "Analyze social sentiment",
    context: {token_name: $token_name}
  )
}
WAIT_ALL

$report = MERGE_RESULTS([$price_analysis, $volume_analysis, $sentiment])
