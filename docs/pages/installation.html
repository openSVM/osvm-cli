Installation Guide

▓▓ Quick Install

curl https://osvm.ai| sh

▓▓ Binary Installation

Download pre-built binaries from GitHub Releases:

# Linux (x86_64)
wget https://github.com/openSVM/osvm-cli/releases/latest/download/osvm-linux-x86_64
chmod +x osvm-linux-x86_64
sudo mv osvm-linux-x86_64 /usr/local/bin/osvm

# macOS (Apple Silicon)
wget https://github.com/openSVM/osvm-cli/releases/latest/download/osvm-macos-aarch64
chmod +x osvm-macos-aarch64
sudo mv osvm-macos-aarch64 /usr/local/bin/osvm

# macOS (Intel)
wget https://github.com/openSVM/osvm-cli/releases/latest/download/osvm-macos-x86_64
chmod +x osvm-macos-x86_64
sudo mv osvm-macos-x86_64 /usr/local/bin/osvm

▓▓ Build from Source

# Clone repository
git clone https://github.com/openSVM/osvm-cli.git
cd osvm-cli

# Build release version
cargo build --release

# Install globally
sudo ./install-release.sh

# Or use make
make install

▓▓ System Requirements

  • Rust 1.80.0 or higher (for building from source)
  • Solana CLI tools (optional, for blockchain operations)
  • Linux, macOS, or Windows (native support)
  • 4GB RAM minimum (8GB recommended)
  • 10GB disk space

▓▓ Verify Installation

osvm --version

Check installed version

osvm doctor

Run system diagnostics

osvm doctor --fix

Auto-fix common issues

▓▓ Configuration

OSVM uses configuration from multiple sources:

  1. Command-line arguments (highest priority)
  2. Environment variables
  3. Config file: ~/.config/osvm/config.yml
  4. Default values

▒▒ Example Config

json_rpc_url: https://api.mainnet-beta.solana.com
keypair_path: ~/.config/solana/id.json
commitment: confirmed
