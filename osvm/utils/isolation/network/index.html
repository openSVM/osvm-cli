Zero-trust networking layer with mTLS

This module implements mutual TLS (mTLS) authentication for all component communication. Every network connection between components requires:

  • Valid client certificate
  • Valid server certificate
  • Mutual authentication
  • Encrypted channel (TLS 1.3)
